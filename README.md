## 概括

基于Gin框架和MySQL数据实现的一个问答系统

当前已经完成的功能

1. 新用户注册
2. 用户登录，同时在之后的操作中进行鉴权
3. 用户登录后可以提交自己的问题
4. 登录后可以对相关问题提出自己的回答
5. 登录后可以使用搜索功能通过关键词对问题进行查找
6. 可调用大语言模型对用户的问题进行回答

当前仍在开发的功能：

1.用户可以对自己提出的问题和答案进行删除和修改

2.点赞与点赞排行

3.增值功能
